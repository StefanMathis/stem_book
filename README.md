# stem-book

This repository contains the manual for the STEM (Simulation Toolbox for
Electric Motors) ecosystem - a framework for simulating electric motors written
in Rust. It is hosted here:
[https://stefanmathis.github.io/stem_book/](https://stefanmathis.github.io/stem_book/).

The manual is written in [Markdown](https://www.markdownguide.org/) and is built
using [`mdbook`](https://rust-lang.github.io/mdBook/). 

To view the book locally, install `mdbook`. Then, from the root of this
repository, run:

```bash
mdbook serve --open
```

This starts a local webserver and opens the book in it. See the documentation of
`mdbook` for more.
